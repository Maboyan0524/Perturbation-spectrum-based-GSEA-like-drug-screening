# Perturbation-spectrum-based-GSEA-like-drug-screening
This system is based on collected drug transcriptomic profiles or perturbation profiles. By analyzing the regulation direction and magnitude between key disease targets and drug profiles, it quickly screens and ranks the regulatory effects of drugs in the database on the selected targets.
